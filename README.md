> **2024–2025** | [View Repo](https://github.com/Shre477/taskflow)

**What it does:** Kanban-style project manager with real-time task updates, team collaboration, and JWT-secured access control.

**Key Technical Decisions:**
- 🔐 **JWT + bcrypt** — stateless auth with hashed passwords, protected React routes via HOC
